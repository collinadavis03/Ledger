# Screenshots

Add your screenshots here before pushing to GitHub.

**How to take them:**
1. Run the app locally with `npm run dev`
2. Sign in and add a few transactions
3. Take screenshots of each tab:
   - `overview.png` — the main dashboard with charts
   - `budgets.png` — the budgets tab with progress bars
   - `import.png` — the CSV import screen

**On Mac:** Cmd + Shift + 4, then drag to select the area

Then update README.md to reference them:
```markdown
![Overview](screenshots/overview.png)
```
